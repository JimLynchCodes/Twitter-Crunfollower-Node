# Twitter-Crunfollower-Node
A node script meant to run as a cron job that unfollows someone you're following who isn't following you back.


## Usage

Please use node v13.13.0
```
nvm use
```

Install dependencies:
```
npm i
```

Run the script:
```
npm start
```
